# 🧩 PokeAPI Explorer

Aplicação web que consome a **PokeAPI** para buscar informações de Pokémons em tempo real.  
O usuário pode digitar **o nome ou o número do Pokémon** e visualizar rapidamente:

- Nome
- ID
- Imagem
- Ataques principais

Este projeto foi desenvolvido com foco em **prática de consumo de APIs, manipulação do DOM e programação assíncrona com JavaScript**.

---

# 🚀 Demonstração

A aplicação permite:

1. Digitar o **nome ou número do Pokémon**
2. Buscar dados diretamente da **PokeAPI**
3. Visualizar informações básicas do Pokémon
4. Exibir **imagem e ataques**

---

# 🛠️ Tecnologias utilizadas

- **HTML5** → Estrutura da aplicação  
- **CSS3** → Estilização da interface  
- **JavaScript (ES6+)** → Lógica e manipulação do DOM  
- **Fetch API** → Requisições HTTP  
- **PokeAPI** → Fonte dos dados dos Pokémons
